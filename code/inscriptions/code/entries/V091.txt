请虔诚弟子炮放在宫桌上，由宫集体放，为了看人安全。<br><br><br><br>DSC02880/2882-2883<br>                台星宫简概史<br>    台星宫（俗称后院宫）是属于后院、后许、岭顶及钱塘郑氏的总宫。据古人<br>传说和历史参考，台星宫创建至现在已有四百四十七年历史。<br>    后院村在明朝嘉靖1547年朱文汉号叫科考入进仕官居南京事中。当时御赐<br>木坊建在钱塘灵应宫前40米左右。同侄朱彬在一五四七年考入进仕官居广东府<br>通判还请旨回乡建御赐石坊建在黄石塘头，在嘉靖太岁庚戌年公元1550年，同<br>时领旨回乡创建如星宫。一五五O年座落在旧宫地，座向东宫门与谷城宫宫门<br>相对，结构双座厝，对中有天井。台星宫建成后本宫社丁昌盛，人物平安。至清<br>朝后来风景（风水）被破坏，地理位置变更等原因，各方面不比以前好。所以会<br>同同亲于中华民国十一年太岁壬戌年公元一九二二年迁建台星宫到现在新宫地，<br>就原旧宫地向西移100米左右，座向南，结构一厅二房，至到公元一九六O年<br>共和国开展社会主义教育，政府号召破四旧立四新。台星宫拆毁平地后，继文化<br>大革命开始，提出不准建宫庙故此放空地二十八年。等到一九八八年在三村乡老<br>公和董事会重新发动虔诚弟子题缘重建台星宫一座，座向南，结构一厅二房。<br>    一九九七年因国家改革开放促进国民经济发展。建设高速公路经过台星宫需<br>要拆迁，当然要服从国家大局为重，共同协商之下按原来宫向东移10米左右，<br>座向变动向西，就是现时迁建台星宫，面貌焕然一新。其中一厅二房是由人民政<br>府负责赔偿迁建，是在镇拆迁办及村的主干努力支持分不开。总补助人民币<br>47500元（包括迁宫，征地赔果树等），迁建由三村乡老公及董事会负责，发动<br>虔诚弟子题缘。宫杉料换大，扩大两边小厅。<br>
