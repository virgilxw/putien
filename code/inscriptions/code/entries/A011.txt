011  章江八社<br>   章江八社地处罗外大沟与洋城斗门前大沟之间，原属于南力里的樟江铺。当<br>地明代有章鱼头港和涵口二村，清代分出港利、西厝、后周、后彭、北厝、尾厝<br>等村，近代分出前湖、后湖、灰壁等村，俗称“港利十八墩”。现有港利、港西<br>二行政村，总人口约3400人。<br>   本地彭姓和周姓为科举望族。彭姓现有人口约2000人，族谱、祠堂尚存。据<br>说莆田彭姓自港利涵口发衍，黄石横塘、城厢南箕（张镇）及同安等地的彭姓曾<br>来此认宗。先祖彭韶，明天顺元年（1457）进士，官至刑部尚书；彭鹏，清顺治<br>年间（1644-1661）举人，官至广东巡抚。明弘治元年（1488），巡按御史为彭韶<br>立“大司寇”牌坊。明弘治七年（1494），兴化知府为彭韶父子里“世科”牌坊。<br>周姓现有人口约500人，在明代有1进士7举人。明景泰元年（1450），兴化知<br>府在章鱼头港为周埮立“进士”牌坊。<br>
