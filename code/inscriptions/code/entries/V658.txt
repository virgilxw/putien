寿海庙从前是奉祀齐天大圣为主，于清朝乾隆伍叁年，美尾村发生时疫，村民忧心忡忡，到处求神问佛，祈求平安。传说先由五更早下海捕鱼，村民眼见一秀士头上清朝官典，身着白袍，俨然端坐庙前海堤之上，继有多数海边居住村民在初一和十五之日更深夜静之时，突然然听到海上有船只来港喝令抛锚下帆停泊船只动作之声，有的村民经营海货及负肩挑搬运者起床赶到码头，静悄悄连一只船影都没有，受惊毛发疏然，急逃回家，紧闭门户，故此公众召集有关人等到庙请筊、请砧，求示批语，才知道是十多年前联呈弹劾大奸妄和绅祸国殃民各种罪行，只因和绅是乾隆得宠奸相，不但拔凶不倒，联呈同科欧田马郭四进士无辜蒙难，轰动全国，各地百姓无不为四位忠君爱民青天为民殉职痛心流泪，所以在民间封这四位进士，多有立庙纪念，降临美尾村，灵砧批示玉封郭公圣侯并任都察院巡察御史，乡人祈求郭公保佑驱走厉疫，自愿立庙奉祀，自此以后，果然疫散全乡安谧如恒，乡人感其恩德，推选杨仲增为功德主，负责劝募扩大修建寿海庙，于乾隆伍叁年扩建修理如现状，以奉祀郭爷为主，时经二佰多年，亦经多次重修，至公元八七年，庙前公路逐渐加高，庙宇无形下沉，平时庙中积水，孳生白蚁，经乡里热心公益事务人士会同旅星华侨共同集资重建，经郭公圣侯灵筊批示高低宽窄，结构一律如旧，平地加高壹米，虽然工程浩大，费用亦巨，在全乡群众、海外华侨群策群力推动下，四月初一拆庙清基开始动工重建，同年八月完成一切工程和设备，值逢郭公圣侯寿诞举行开光告竣，今为永远纪念集民间传说及重建过程写成庙史，以资志念。<br><br>
