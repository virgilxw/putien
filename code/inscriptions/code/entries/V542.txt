古建筑是国家重要的历史文化遗产，是国家文明的重要标志。根据《中华人民共和国文物保护法》的精神，为加强消除管理工作，特制订以下几点：<br>    一、国家公共财产是我国公民的神圣义务，古建筑的消防工作，要贯彻从严管理，防患未然的原则。<br>    二、防火灾列为整个管理工作的一个重要部分，切实做到同计划、同部署、同检查、同总结，同评比，使防火工作做到经常化，制度化。<br>    三、管理会主任是法定的防火安全负责人，全面负责单位的消防安全工作。<br>    四、禁止在本建筑保护范围内堆存柴草、木料等易燃可燃物品。严禁利用将煤气、燃化气等引入本建筑物内。<br>    五、在厢房、走廊、庭院等处需设置生活用火时，必须有防火安全措施，并报请上级文物管理部门和当地公安机关批准，否则一律取缔。<br>    六、本建筑保护区的通道、出入口必须保持畅通，不得堵塞和侵占。炽前池塘是原有的天然应妥维护，保障消防用水。<br>    七、安全管理范围：南至戏台，北至果园，东至民房，西至民房，南北三十三米，东南宽一十七米。<br>    八、公民在发现火警时，应迅速报警，并立即进行扑救。<br>    九、在预防火灾中工作积极，成绩显著，在灾火战斗中英勇机智、表现突出，使国家财产免受重大损失者，应予表彰和奖励。<br>    十、对防火工件工作放胜地自流，玩忽职守，以引起火灾，使国家财产遣受损失者，应据情节轻重，由主管部门予以行政纪律处分。或由公安司法机关依法查外，直至追究刑事责任。<br>
