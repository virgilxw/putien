码头<br>    原称榜头，包括留山（刘山），属呈山行政村。现有人口约1000人，有朱、<br>陈等族姓，朱为主要族姓。<br>    主要庙宇：<br>    榜头境登荣新社，主神为尊主明王、后土夫人，陪神为天上圣母、杨公太师、<br>司马圣王。<br>    元宵仪式：<br>    1．13-14，请登莱书社诸神巡游本村，演戏，请道士做醮。全村统一组织，<br>按丁口集资。<br>    神诞庆典：<br>    2、25，参加呈祥宫齐天大圣巡游。<br>    3．23，天上圣母诞辰，演戏，请道士做醮。全村统一组织，按丁口集资。<br>    4．26，杨公太师诞辰，演戏，请道士做醮。全村统一组织，按丁口集资。<br>    8月卜日，司马圣王诞辰，演戏，请道士做醮。全村统一组织，按丁口集资。<br>    10．13，尊主明王、后土夫人诞辰，演戏，请道士做醮。全村统一组织，按<br>丁口集资。<br>    <br>
