132 涵江二十四铺<br>    涵江地处木兰陂、延寿陂、太平陂三大流域的交汇点，为水心河和端明斗门所在地，原属于延寿里的仓头沙阪铺、前后街铺、青宁铺、孝义铺、鑑后铺、乌台忠孝铺、楼下铺、新桥头铺、下徐铺、宫下铺、西园后度龙津铺、新牌铺、延宁铺。当地明代已有涵头市和后度、塘北等村，清代发展为繁华的港口市镇，现已全部改造为街区，分属于涵东、涵西街道办事处。在神庙祭典中，主要按传统铺境组织，俗称二十四铺。不过，现存庙宇及仪式传统较为完整的社区，只有楼下、保尾、塘北、铺尾、苍然、延宁、霞徐、林墩等边缘聚落。<br>    位于原鑑前铺的鲤江城隍庙为涵江市区二十四铺的中心庙宇，始建于清康熙五年（1666），主神为城隍大神。每年农历正月十九举行二十四铺总元宵，五月十九举行城隍诞辰庆典，农历七月十五和十月十五举行“祭孤”仪式。每逢农历闰五月，请城隍大神巡游涵江街道及附近村落，由二十四铺代表组成的董事会统一组织。<br><br><br><br>DSC04261<br>公  告<br>    为祈祷世界和平，实现平安涵江，暨庆祝本庙重兴二十周年庆典，谨择定：<br>太岁乙酉（二00五）年农历二月十二日举行叩祭旗杆、立杆安装法事。三月初一日天上圣母升座行台。三月十五日开坛大醮，虔修延生锡福道场，组织彩架、车鼓、十音、八乐等大型精彩民间文艺踩街活动，随驾天上圣母巡游赐福。<br>    恭逢上列吉庆良辰、恳望善信人众来庙瞻仰朝拜，同沾福祉。<br>    谨具公告<br>    风调雨顺    国泰民安<br>    涵江城隍庙管委会<br>    明代涵头市为科举名区，全镇14姓共有17进士64举人。其中陈姓有5进士16举人，黄姓有4进士11举人，林姓有1进士7举人，郑姓有1进士5举人，李姓有2进士3举人，张、方二姓各有4举人，刘、卢二姓各有二举人，朱、高二姓各有1进士，徐、蔡二姓各有1举人。明正统十一年（1446），为陈姓立“进士”牌坊。明正统十三年（1448），为林姓立“贞节”牌坊。明景泰七年（1456），为邱姓立“应奎”牌坊。明成化二年（1466），立“涵江书院”牌坊。明成化三年（1467），为陈姓立“步玉”牌坊。明成化十三年（1477），为陈姓立“奎璧联辉”牌坊。明成化十九年（1483），为陈姓立“贞节”牌坊。明弘治十二年（1499），为邱姓立“进士”牌坊。<br>    在乌白祺械斗中，延宁、保尾等铺为乌旗，下徐铺为白旗。<br><br>
