012  新度九社<br>   新度九社地处上下渠大沟与罗外大沟之间，原属于南力里的九桥铺。当地明<br>代有新度村，清代分出九桥、东店村。现分为后亭、度边、东店、三角曾四大聚<br>落，而神庙祭典主要分为“九社”组织。<br> 本祠地处莆田新度村中部，居城黄公路之间南侧，座北朝南，1059年创建。<br>始祖为河南省因始县朱阜里内汤史瀚公之次子文立公，立之子魁秀公，号真庵，<br>唐太（大）和（文宗皇帝李昂）四年（830年）举孝廉，历宫泉南太守，遂居莆<br>田城聚庵巷，创建祠堂，后名“汤堂巷”，为汤氏入莆之始祖。<br>    第三世学敏公子唐大中（宣宗皇帝李忱），847—859年间举进士，官居刑部<br>员外郎。第五世汤？公于后唐同光（庄宗教皇帝李存勖）约923—925年任秘书<br>省校书郎。第六世汤庆公任广东韶州府（今韶关）主簿，遂家焉。第七世汤梧公<br>任南郡（福建）参军功封招使。<br>    第八世汤淇游学江南，居安徽省歆县；汤？公于北宋嘉祜（仁宗皇帝赵祯）<br>三年（1058年）举进士，官居司马。始迁九桥新度村，淑配林氏瑞伯公女，葬<br>新兴里（即赖溪桥前近西放村公路边），坐甲向庚，侧有亭，名曰“汤亭”。1059<br>年创建“汤氏祖祠”。<br>    第十一世汤曰瑚公之长子文用公于南宋始定（理宗皇帝赵昀）次年（1229<br>年）兴进士，官居常州太守（今江苏省常州市）、次子文学公于南宋嘉定（宁宗<br>皇帝赵扩）首年（1208年）任广东提举，逐居广东省增城市（今分居入城汤、<br>坡吓汤、北门汤、新汤等，总称为北门汤）。第十三世汤则翼公游学福州水巷，<br>后迁福清、汤则艮任兴化军授司卢参军。第十五世汤廷优公于元贞次年（1296<br>年）举进士，任刑部郎中，教谕公女、汤廷选公任芦溪（今涵江区秋芦镇）主簿。<br>第十七世汤亨公施造南寺（即广化寺）塔浮图一层。<br>    第十八世汤宜昌公之子用中公，于明洪武（太祖朱元璋）十三年（1381年）<br>举经明行修补国子监五经博士（次于进士），生明德、教德、竣德三子，为新度<br>九桥汤氏天地人三房。明德公生希移、希圣、希仁三子。孝德公生希尧、希舜两<br>子。峻德公生希稷。从此开六房世系；宜显公之子用公于元至正十年（1350年）<br>设教兴化县（今涵江区新县镇）大所，遂家焉，为大所汤氏始祖，生一子，名曰<br>道渊：宜晋公之子用时公于元至正十六年（1356年）迁往山里（今涵江区秋芦<br>镇）南坛村，为南坛汤氏始祖，生道虞、道进、道秀、道统四子，二房道进迁福<br>清溪底；宜晟公之长子用行公入赘苏州陆家、次子用光公庙教仙游遂家焉。<br>    始祖入莆迄今1174年，已传42世，在世见面的已有“七代同堂”，族繁人<br>茂，子孙蕃衍各地，可称旺族之邦，人才辈出。<br>    当代汤氏更是后彦云集，英才满天。专家学者、研究生、大学生，不胜枚举，<br>士家工商，科教文卫，各有贤达，事业有成者，数不胜数。<br>    建祠距今已有945年，历史悠久，宋、元、明、清、民国几经修葺，幸得几<br>立于世，激扬时代气息，显得古朴而恢宏。1958年被他用，祠门上方匾额，主<br>梁高悬巨幅“文魁”、“亚魁”、“迄魁”、“进士”等族贤祠匾文物尽失，祠宇破损<br>严重，陈旧不堪。<br>    1983年由族人献工、捐资贰万多元，以保护文物为原则进行按原貌修建。<br>    1996年因扩宽城笏公路，改造村房，宗亲同心协力，踊跃投劳，热情乐助，<br>主动捐资壹拾伍万多元，按原方向，移前32社，石砖木结构，施工扩建，规模<br>胜昔，占地面积488平方米，建筑面积241平方米，宗祠大门为三门六扇开，上<br>方悬挂“汤氏祖祠”横匾，祠前石柱对联“官居五马荣东野，位列三台耀中山”。<br>官员人分上下厅，中间过水天井，上厅六石柱，主厅正面设神主龛，供奉历代先<br>祖神像。群策群力，很快完成扩建工程，大所、南坛、月坪、柳桥、广东省增城<br>市北门汤等宗亲分别于1997、1998年莅临本祖祠庆典，拜祖，并题贵匾悬挂在<br>主梁上，提高祖祠品位和文化艺术色彩，成为汤家返思先祖，展示贤名风彩之胜<br>地。今日序碑似斯蛰蛰作述重光，他年为官如瓜瓞绵绵后生辉映。<br>
