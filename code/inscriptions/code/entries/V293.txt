    西亭<br>    一、村落<br>    属东宋行政村，现有人口约380人，主要姓氏为陈、蔡、曾，其中蔡姓约占<br>全村的2/3。<br>    二、族姓<br>    蔡姓分顶厝、下厝二房，人数两房差不多，陈、曾两姓在分布上介于顶厝和<br>下厝之间，主要靠养家猪（数量不多）。<br>    三、村庙<br>    金山宫，修建有50-60年。主神：田公元帅，陪神：杨公太师、刘公元帅。<br>    麒麟东社，主神：尊主明王、后土夫人。<br>    元宵仪式：<br>    1、11-1、12，抬田公元帅巡游本村。<br>    神诞庆典：<br>    4、9，田公元帅诞辰，演戏二天。<br>    仪式团体：<br>    每年按年龄（中年人、老年人没有主持）轮流2人做头主持，没神童，请经<br>师3人。<br>    神诞庆典，请经师1人念咒，部分有演戏。<br><br>
