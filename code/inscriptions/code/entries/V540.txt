渭阳<br>    渭阳三甲地处紫霄山麓，主要利用明厝坑圳、双义坑圳的水源，原属于兴教里的东张山后铺。当地清代有围庄村，现有人口1100人，分为尾厝、半山、石牌兜三大聚落，统属于渭阳行政村。在神庙祭典中，主要按聚落分为三甲。职业以农为主，盛产龙眼、枇杷。<br>    本地为杂姓混居之地：居民尾厝以黄姓为主，另有林姓10多户；半山以陈、郑为主，另有赖、周等姓；石牌兜有钟、陈二姓。渭阳黄氏在清朝由泉州安海迁来，已传到第11代，晚清时是涵江最大的商人家族。原有义兴宗祠、瑞裕宗祠，已荒废。<br>    在乌白旗械斗中，渭阳三甲皆属于红旗，但偏向于白旗。<br>    主要庙宇：    <br>
