妈祖姓林氏名默，世居莆田澄渚岛石系“九牧”第六房蕴公七世孙女，高祖林园，五代时仕*，曾祖保吉仕后周*德元年（934）任统军兵马使，*于天下纷乱，亲官归隐*良港，祖父名系官福建总管，父亲名林愿宋初官都巡检，母王氏生二（名洪毅）六女，妈祖是最小一女。妈祖诞生于宋建隆元年（960元）三月二十五日。在她出生之前父母已生五女，盼望再生一男。因而朝夕焚香祀天，祈求早赐麟儿。可见这胎又是女婴。父母大失所望就在婴儿出生傍晚邻里乡亲看见流星化为一道红光从西北天空射来照耀乌上岩石发红。父母感到这女必非等闲之女，生至满月不开哭声，便取林默。处幼聪慧，七岁教学文义，十岁喜静坐一室，十二岁有老道者玄遣受以玄微秘法，十五岁能为人治病，常渡海至湄洲救助海上遇险船舶。因而人们传说她能“乘席渡海”她还会预测天气变化，事前告知船户可否出航，所以又传说她能“预知休咎事”，称她为“神女”，“龙女”，她专心致志为慈善公益事业，为人治病，教人防疫消灾，经常引导人们避凶趋吉，助人为乐，其秋九月与兄浮海而北，时西风正急，狂涛震荡，女在机织，忽闭睫，神色有异，手持校，足踏机轴，状如有所挟，母怪其假寐，抚之醒，而梭堕地，泣曰“阿父无恙，兄没矣”，俄而父归，言于怒涛中几溺者屡，似有主其柁令二舟相近者，终于父舟脱险，兄舟柁折而覆，次日女与母嫂驾舟入海寻浔兄尸，乡人神之，称为神姑，二十八岁九月九日别诸姊曰“今欲远游，以畅素怀恨不得同行”，家人漫笑慰之，遂渡海上湄峰最高处，闻隐隐有鼓乐声，彩云布合，遂不复见，时雍熙四年（947）年丁亥，里人建庙于其处祀之。即（今湄洲祖庙），所以，妈祖先祖家是澄渚、乌石，诞生地是忠门贤良港，升天地是湄洲岛。<br> 创建于南宋的西天尾镇龙山村龙山（妈姐）宫，位于风景秀丽的龙山西侧山崖，这里交通便利，环境优美，名胜古迹甚多，是西天尾地区至莆田北洋地区民众信仰朝圣的中心，故有莆田妈姐三大宫称誉，同时她又作为妈祖祖籍地的宗教文化载体，在海内外信徒中享有极高的威望。现已成为荔城区首批文物保护单位。<br>    二十世纪八十年代初在热心人士倡议下，由当地老人处发成立管理董事会，经过二十多年的捐资兴建和精心维修，龙山宫面貌一新。逢年过节，这里都有举行演戏等文化盛事，特别是举行一年一度的妈祖生辰和升天祭拜仪式，吸引四方民众前来参加庆祝活动。龙山宫已成为方圆百里群众会拜、游览和娱乐的好去处。<br> 龙册宫，因其坐落在山的西部，俗称西山宫，位于荔城西天尾镇龙山村境内，坐东北朝西南，距西天尾镇约1公里，离福厦路半华里，水泥村道直通宫前。宫宇倚山就势，整座建筑由正殿、观音殿、梳状阁及耳房廊庑和戏台组成，占地面积1700平方米，建筑面积达650平方米。该建筑历史悠久，是妈祖居地、九牧祖祠重要的文物载体，也是莆田妈祖三大宫之一，受到海内外民众的信仰和崇拜。2004年6月被荔城区列为第一批重点文物保护单位。<br>    龙山宫始建于南宋，明代重修，清嘉庆间重建。大殿面阔三间，两进深，抬梁歇山造。殿内正中台上[祀有头戴九龙八凤珠冠，身穿黄色龙袍，双手扶圭的妈祖林默之神像，与湄洲妈祖、白沙妈祖之神像为孪生三姐妹。相传，他们是同一棵香樟树锯成三段分别雕成的妈祖神像，距今已有近千年的历史。龙山宫每年逢农历三月廿三日和九月初九日为妈祖的生辰和升天日，其盛况空前，场面壮观，演戏之多，成为北洋地区一道亮丽的文化景线。后座为观音殿，楼上祀观音和韦陀、迦蓝等神像。第三座是妈祖的梳妆阁，于二十一世纪建成，整座阁宇雄伟壮观，古朴典雅，尤其是由莆田现代名书法家题写的27体不同风格的书法石碑廊，为龙山宫赋予了新的文化内涵。<br>    通往龙山宫峰顶附近有仙足、石椅、石灶等等遗迹，相传九公款山的仙公曾柱仙杖下山巡视龙山，其驻地蓄水成池，称为仙？。这里有狮子吐雾奇观，这里又是观云海的所在，站在龙山最高处的万福寺钟楼上，环视北洋平原，一览无余，令人流连忘返。龙山（妈祖）宫，以民众信仰者众多，独具特色的自然和人文景观，成为人们观光、休闲和朝圣的好去处，并将吸引更多的海内外有识之士前来投资兴业。<br><br>    <br>    此砚造型奇特，刻工粗放，古朴大方，为唐代原物。据《闽书·方域志》及《八闽通志·人物》等古籍记载，晋江名士欧阳詹唐贞元间谐莆与藻、蕴兄弟共同研读于灵岩精庐、福平书堂，此砚随之处泉山迁移入莆，后披公处北螺迁居乌石，筑澄渚书堂，大石砚遂即置于今址，成为九牧兄弟研墨攻书之文房瑰宝。<br>    砚泉古井与碑刻<br>
