051  笏石三十六甲<br>    笏石原称跂石，地处国清塘畔，清初划为界外，原属于合浦里的笏石铺。当<br>地明代已有跂石村，清代分出大龟、林炳、后井、后黄、陈厝、吴厝等村。自清<br>末以来，笏石已发展为界外地区的中心市镇，目前为秀屿区政府所在地，大多数<br>居民隶属于笏石街道。原有聚落形态较为完整的农村社区，分属于大丘、顶社、<br>北埔等行政村。在神庙祭典上，主要按传统聚落划分为三十六甲。<br>    位于陈厝村的聚仁宫，原为三十六甲总宫（铺宫），主神为玄天上帝，以前<br>不定期巡游所属各境。目前，笏石三十六甲已分化为分别以陈厝的聚仁宫、顶厝<br>的圣仁宫和北埔的凤山寺为中心的三大仪式系统。<br>    在乌白旗械斗中，笏石三十六甲皆属于乌旗。<br>    <br>
