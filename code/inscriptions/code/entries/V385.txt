万兴祖社，历史悠久，名闻遐迩。追根溯源，系始建于北宋开宝年间的一无名小庙宇义基础上迁移繁衍的。是时延寿溪溪流横溢，黎民受害不堪言喻遐思，求神灵保佑。此庙宇便应运而建，供奉社稷之神，祷祝庇荫。庙宇周围地段，便名社兜。这就是社兜的历史由来。<br>    妈祖羽化后，这位急公好义，扶危济困的通灵神女神奇事迹，渐由湄洲传开，深受人们敬重。乡民便在社兜庙宇内安位奉祀。“灵妃一女子，瓣香起湄洲”。南宋时，社兜庙宇遂嬗变成湄洲行宫，香火旺盛，绵延不绝。后迭遭洪水洗劫而衰废。<br>    明嘉靖末期，里人建议：嵋屿行宫，由社兜迁往桥头（今址）易地重建。檀越林有声，林大成，克绳祖武，折腰募款、捐地。历年余，坐东朝西新的湄洲行宫屹立桥头坡上，按干支五行推论，其坐向大煞风水，必殃民伤财，民信其然，越数载，遂改建为今之坐向。万历甲申年，流合桥建成，洚水被驯服，南浦大地咸受其益，生意盎然，合乡同庆。斯时行宫嬗变为万兴祖社，寓万物生灵俱兴之意。今西洋万兴新社，郊南万兴瑞龙洞，郊北万兴新宫，皆由祖社繁衍而来。<br>    万兴祖社建成后，人文蔚起，甲第蝉联，岁韦丰穰，稼穑孳殖。信而有徵。清乾隆三十八年，祖社重修。雕梁画栋，再现金身，并立碑石为志。继往开来，数典不忘其祖，懋赏后嗣。<br>    一九四二年，祖社再次修葺，旧貌新颜，开光告竣时，香烟缭绕，社戏连台，信读膜拜祷念：攘除日寇，免遭兵燹。盛况空前。解放初，民风不古，烟火尽废。一九五八年，又遭特大洪水洗劫，夷为平地。时大兴农业，社址被垦植，碑石受毁，无一幸免。文革，可供考证的族谱之类文记，亦遭厄运，无一存留。一九八O年，祖社复建，香火又兴。<br>    二OO三年，融祖社与行宫为一体的拜亭落成，会合村长老，共计祖社事，集村情民俗，祖社遗风，先人见闻，民间传记，轶闻轶事，碑文族谱，予以谱记论证。撷其精要，书成史略，勒石铭志，以供后人稽考。<br>                        公元二OO三年癸未桂月立<br>
