邓厝<br>    分上邓、下邓二甲，邓为主要族姓。<br>    主要庙宇：<br>    沁后境福兴社，道光辛丑年重建。主神为尊主明王、后土夫人；陪神为五谷仙（神农氏）、田公元帅、张公圣君。<br>    元宵仪式：<br>    1．15，请社炉到各户行道，晚上绕境游灯，请道士做醮。<br>    神诞庆典：<br>    4．29，尊主明王、后土夫人诞辰，演戏3-6天不定，本社主持。<br>    5．15，五谷仙诞辰，由和平四甲轮流，各演一天戏。<br>    8．13，田公元帅诞辰庆典，演戏2天左右，由牛山主持。<br>    演戏戏金按丁摊派或自由题缘。<br>    五、其他：<br>    无神童、京师，有一个师公，乃世代相传。<br><br>
