147  江口七境<br>    江口七境地处江口镇附近，主要利用南安陂支渠的水源，原属于待贤里的孝义坊舖、亲仁舖和锦绣坊舖。当地明代有孝义、港下、前会、后墩、刘井、度围头等村，清代从后墩分出新井，民国从孝义分出后度，现有八村。由于后度仍参加孝义佑圣观的元宵巡游，因而只有七个基本的仪式单位。<br>    江口七境原有总宫“真君殿”，主祀三殿真君及文昌帝君、魁斗星君，后改建为锦江中学，目前正在筹划重建。以前每年正月元宵节期间，都要请三殿真君巡游七境，东岳观出游亦必驻驾于此。目前尚未恢复元宵总巡游仪式，但每年六月初十真君诞辰，仍演戏以示庆贺。<br>    江口前会村尚存“兴贤祖社”，据说始建于宋代，为七境“古社”，乾隆52年重建，近年重修。每年二月初二和十二月初六的“头尾牙”，江口七境的每户居民都来祭拜。<br>    江口七境原有三十六姓，各姓都是聚族而居。现仅存十五姓，各村亦多为杂姓混居。不过，在村庙的仪式组织中，族姓仍是主要的认同标志。<br>    在乌白旗械斗中，江口七境所属各村皆为白旗。<br><br>
