坊井<br>    原称杭井，属北高行政村，现有人口600多人，有高、吴、陈等姓，高、吴<br>为主要族姓。<br>    主要庙宇：<br>    群英书社，主神为尊主明王、后土夫人，陪神为齐天大圣、吕洞妈、福德正<br>神。<br>    元宵仪式：<br>    1．12，请吕洞妈出宫到福首家祈福；<br>    1月15举灯<br>    1、16，圈灯，请道士3人做醮；<br>    1、17放鞭炮。全村统一组织，按丁口集资。<br>    神诞庆典：<br>    2、25．齐天大圣诞辰，演戏，参加呈祥宫齐天大圣巡游七境。<br>    6．6，福德正神诞辰庆典，演戏，请道士做醮。全村统一组织，按丁口集资。<br>    8、23，尊主明王、后土夫人诞辰庆典，演戏，请道士做醮。全村统一组织，<br>按丁口集资。<br><br>
