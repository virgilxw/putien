036  员永五村<br>    员永五村地处后洋大沟，原属于连江里的阜墙员永铺。当地明代有上余、下<br>余、石阜、象峰等村，清代分为五龙、新安、度尾、沟边、长坑、下余等村。现<br>有人口4000多人，分属于水南、和平、凤山三行政村。<br>    位于长坑村的天心台为五村总宫，主神为田公元帅。每逢丁年，卜日请田公<br>元帅巡游五村，由五村代表组成的董事会统一组织，所属各村按既定比例分摊费<br>用：沟边40%、五龙30%、道美19%、长坑8%、下余3%。<br>    在当地历史上，余、陈、吴、黄、徐、王、邱等姓为科举望族。宋绍熙年间<br>（1190-1194），兴化知军为余振立“金紫”牌坊。明宣德五年（1430），兴化<br>知府为陈中立“会元”牌坊。明天顺四年（1460），兴化知府为徐姓立“梅陇世<br>科”牌坊。明成化十八年（1482），莆田县丞为黄姓立“旌表贞节”牌坊。明成<br>化十一年，福建布政司为吴姓立“亚魁”牌坊。明成化二十二年（1486），巡按<br>御史为王姓立“步蟾”牌坊。明弘治十二年（1499），巡按御史为邱姓立“繍衣”<br>牌坊。明弘治十三年（1500），莆田知县为陈元立“登科”牌坊。<br>    在乌白旗械斗中，员永五村皆为白旗。<br><br>
