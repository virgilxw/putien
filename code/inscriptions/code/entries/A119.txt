119  碗洋四社<br>    碗洋地处紫宵山麓，主要利用官塘、澄塘的水源，原属于兴教里的碗窑后黄铺。当地清代有碗窑村，现有西洋、后洋、隔垄、田厝等村，统属于碗洋行政村。在神庙祭典中，主要按社庙系统分为四社。<br>    位于西洋村的关圣庙为四村总宫，主神为关圣帝君。每年农历二月初二举行头福庆典（总元宵），请关圣帝君巡游各社。<br>    本地居民主要有施、黄、陈三大姓，另有翁、林等小姓。据说，最早到此是元朝时的黄斌卿。施姓于明朝中期从晋江迁到洞湖，再由洞湖迁来，当时地点叫田头厝，分二大房，其中头房再分4房，二房现分5房，现有人口约300多人。黄姓从澄渚村东黄迁来，现有人口约400人，主要分布于后洋，分四房，原有祠堂。陈姓现有人口250人左右，分为六房，分布于西洋、隔垄各三房，原有祠堂，现为纸箱厂仓库。<br>    在乌白旗械斗中，碗洋四社皆为红旗。<br><br>
