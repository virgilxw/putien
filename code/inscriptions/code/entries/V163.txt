先父维干公，号铁苍，字柏青。1893年生于金山村，村中第一位大学毕业生<br>1992年逝于福州，乃百岁老人也。公生前任福建师范大学历史系教授，研究生<br>导师。并兼任福建省文史委员会委员、福建省史学会顾问、福建省方志编委会顾<br>问、福建省地名委员会顾问等职。公执教达七十个春秋，成绩卓著，硕果累累。<br>是我省深爱敬重的一位年高德劭、学识渊博，为人师表，享誉海内外的教育家、<br>史学家、地方史专家。公年过九十后极盼望能回金山安度晚年，叶落归根，未果。<br>现为了纪念他，经当时尚在世的长子启宇、三子启炎（薛雷）和坚等商定，在祖<br>居地金山建一纪念性小楼，定名怀贤堂，资金由炎、坚分担。兴建过程中，先得<br>到公之堂侄孙祖应之亲属支持提供宅地，后又得坚之同窗挚友莆田市建委总工程<br>师翁春辉、工程师彭文堂、邹元铸等同志的热心而又无偿地为之设计。公之堂侄<br>孙祖珍无私地出任工程总负责人，公之堂侄启浦、孙祖集、侄孙祖奕及普侄孙炳<br>粦、曾侄孙朱岳等亦鼎力相助。村党政领导和福建中医学院旅港校友会会长苏晋<br>南教授等坚之亲朋好友亦给予亲切关怀和支持，工程历时一年，1999年末竣工，<br>怀贤堂终于拔地而起。我们深信，子孙后代必能兴起怀念先贤之风，继承与发扬<br>公之为学、为人、为师之风范，多出人才，出好人才，奋发图强，为国家和人民<br>多做贡献，做大贡献。<br>    最近，重建怀贤堂之大门，全由公在国内外的多位亲孙负责。公在英国的企<br>业家朱祖华孙、叶对妹孙媳领头，公之孙辈朱枫、何小雪、朱霆、张秀琴、朱红、<br>吴东辉、邱鹭航、柯炳粦等主动集资助建。并经由公之堂侄孙朱祖发主动无私地<br>担当起重建的全部工程。公之后裔能如此热心，实在十分感人！<br>    我们全家从内心里感谢为大门题写“怀贤堂”牌匾的原莆田市市长、现中共<br>宁德市委书记陈少勇同志，是他给了我们极为难得的关怀和永远的鼓励！<br>    特此志之。<br>                    福建省第八届人大常委<br>                    福建中医学院原党委书记<br>                    福建省闽台高校交流促进会会长<br>
