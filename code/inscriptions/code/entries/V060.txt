四宫位于新度村西部，座北朝南，规模颇为宏伟，宫貌壮观。南望壶山原形<br>千峰竞秀，东观兰水清百舸争流，西畔木兰荔树成林，北近市场风华胜地，前面<br>风火围墙，风光秀丽，诚乃钟灵福地。<br>    本宫是由原宗孔堂三一教主（旧址在下埕，今村部前公路上），座北朝南，<br>前身名曰“日应堂”，由清光绪监察御史江春霖亲笔题写“宗孔堂”横匾，实属<br>珍宝；月塘祠也是三一教主（旧址在西湖），座东朝西；聚奎书院文昌帝君（旧<br>址在帝君祠，今汤桥头西侧沟边），座东朝西，传说曾有仙人憩息聚奎书院，故<br>有“聚奎院里神仙至，朱雀桥边士子游”的雅话。因此当时本村名曰“仙渡村”，<br>（今有桥碑牌为据），后改为“新度村”；万灵宫孔府大人（旧址在大前埕，今中<br>山路中东侧公路上），座北朝南，民国十二年创建。<br>    上述堂、祠、院、宫由于斗转星移，年久失修，又遭洪水冲垮，以致榱题瓦<br>解，成为废墟。<br>    为了弘扬林龙江先生爱国爱民的优良传统，颂扬尊师之道，除妖去邪，护国<br>庇民，履行宗教信仰与社会主义社会相适应的政策。董事会顺民意，体民情，于<br>1984年选址在壶山墩，将宗孔堂、月塘祠、聚奎书、万灵宫综合兴建在一起，<br>取名为“四宫”。1985年落成，计用人民币16000元。<br>    涌坛武社等主明王（旧址在西湖）座北朝南，经百年风霜，摇摇欲坠，由于<br>经济条件所致，于1958年倒塌。巧逢四宫落成之机，托建在四宫西侧，并重塑<br>神像。<br>    1998年全面进行重建，画栋雕梁，流光异彩，全面油漆，墙壁粉刷，金碧<br>辉煌。柱联、横匾、案桌、龙桌俱全，宽阔石埕完整，大拱门上方飞檐翘角，大<br>理石金字“九桥四宫”匾额，气势雄伟，焕发新姿，更呈辉煌。以志感怀殷念，<br>自此欣逢节日必行，朔望必拜，日日香火鼎盛。<br>    蒙广大热心人士，村民和乡邻舍地、献工、鼎力资助，计捐资260000元，<br>占地面积1254平方米，建筑面积824平方米，2004年梅月工程圆满竣工。是年<br>阳月廿日隆重地举行告竣典礼。捐资刻碑，流芳千古，敬祈弟子寿而富，炽而昌，<br>长发其祥。<br>    撰文：汤锦荣  郭文滔<br>
