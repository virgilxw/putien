西山<br>    全村约230人，杨为主要族姓，另有戴、蔡等姓。<br>    主要庙宇：<br>    万安社，光绪三年建，1993年重建。主神：尊主明王、后土夫人；陪神：杨公太师、普济圣侯、齐天大圣、田公元帅。<br>    元宵仪式：<br>    1．11，到龙山宫进香，请天上圣母神像回村。<br>    1、12，请天上圣母和万安社诸神巡游本村，演戏，请道士1人做醮。本村组织，按丁口集资。<br>    1、13，送天上圣母神像回龙山宫。<br>    神诞庆典：<br>    3．23，天上圣母诞辰庆典，在龙山宫演戏，请道士做醮。龙山宫组织，本村至少负责一天，按丁口集资或自由捐资。<br>    9．17，尊主明王、后土夫人诞辰庆典，演戏，请师公念经。本村组织，按丁口集资。<br><br>
