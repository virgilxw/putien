苍店亭潮云阁，供奉大慈大悲、救苦求难的观世音菩萨，数百年来，甘露普<br>施四境，福泽遍布八方，善信有求必应，香火络绎不绝，众口皆碑。昔时曾有佛<br>塔一座，虽是砖石结构，造作简单，亦是禳灾、驱邪、祈福之镇亭至宝，可惜因<br>世事沧桑，毁于六十年代。八十年代，本亭复兴后，我佛慈悲，普渡众生，施福<br>泽于万民百姓，众善信荷佛恩、感佛德，倡议在本亭之东南一隅旧地，重建镇亭<br>宝塔，以禳灾、驱邪，赐福四境八方，缔造平安莆阳大地，籍宝塔之威灵，超度<br>十方佛子、水陆孤魂，弘扬佛门慈悲、普渡众生。<br>    近日，本亭众董事顺民心、感佛性，同心赞举募缘重建苍店亭佛塔，设计塔<br>高10.6米，塔肚径3米，五层八角花岗岩结构，塔檐翘角，佛像人物填肚，甚<br>是宏伟壮观。预算耗资拾陆万元，将在近期内破土动工。为此，特文告善男信女，<br>望能广施善缘，助建苍店亭佛塔。<br>    募缘方案如下：<br>    一是广募“万人缘”。不论妇男老幼，一分钱也是缘。广种善缘收善果。<br>    二是喜题乐缘，多善乐施，凡缘达壹佰元以上者，芳名刻碑纪念，达伍佰元<br>以上者，芳名刻于塔身碑肚。与塔共存，流芳万古。<br>    三是分层募缘。可一人或数人合捐建一层。并在该层刻上芳名，永志大德。<br>塔身分类五层，除第一层（底层）外，第二层缘金标底3.8万元，第三层缘金标<br>底2.8万元，第四层缘金标底1.8万元，第五层缘金标底8千元。供各选择。<br>
