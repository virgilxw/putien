上林<br>    一、村落<br>    全村现有人口140多户，共750人左右，以农耕为主，辅有果树（荔枝）、捕鱼为副业收入。<br>    二、族姓<br>    据说原来吴氏是上林村大姓，以烧陶为业，后迁居塘头谭尾村，另有林姓也迁徙外村。目前，翁氏占全村人口90%以上，另有黄姓数户、郑姓1户、陈姓1户，吴姓仅存孤老一人。翁氏来自北梧塘东坡村，据说其祖先于200多年前与上林村姓林的女子结婚，生二子，长子姓林，早夭；次子姓翁，留居上林，遂为本村翁氏始祖。翁氏现分为福、禄、寿三大房，福房约30户、禄房约80户、禄房约25户。翁氏祠堂尚存，奉祀翁氏始祖季华公、二世祖诚厚公及三世祖笃心公、养心公、汀心公，各房支在祖厅祭祀三代祖。<br> 一、夏教必须恪遵时王制度，办理宗教社团登记手续，服从党和政府的领导，遵守国家的政策法令，开展正常的宗教活动。<br>    二、坚持入教自愿的原则，凡要求入教者应履行入教手续，举行入教仪式。入教后务须按照“启文”要求，勤行修炼教主的九序内功，不参与旁门邪道的任何活动。<br>    三、遵循正道，修身养性，切勿谈妖说怪，跳神念咒。断人生死，判人吉凶。书符却病，有损“心*性病”之学，更应严禁不正行为。采取以上手段到处招摇撞骗，诈取群众财物者，应受教规严肃处置，严重者要清除出教，在本教内帮忙宗教活动，不得收取酬重厚礼。<br>    四、应以纲常四业为立本，切勿好逸恶劳，不务正业，不参与嫖、赌、酗酒等有损夏教威信的行为。<br>    五、加强团结，同道者应尊老爱幼，谊同兄弟姐妹，有善相规，有过相劝，疾病相扶，患难相救，切勿尔虞我诈，面是背非，挑拨离间，造谣生事，诬师败教。<br>    六、维护夏教的尊严一致，切勿拉帮结派，各树旗帜，违反夏教宗旨，自行其是。<br>    七、夏教祠（堂）应是谈经演教的清净场所。不许在堂内设赌，无故聚众暴饮，言入是非，谈论荒诞不经，乱伦秽亵之事和随便谈论社会上的流言蜚语，有沾祠堂清静，损害教风。<br>    八、凡遇朔望五、十必须组织学习，讲经论道（印证心法）不得散处闲谈，务宜齐集，一所随意取经书一部，互相讲解，不惟可以印证吾身，而心身性命亦可以渐进高明。<br>    九、要定期举行会道、设教度人，讲章内容应以夏教经典著作为主，劝善文，因果报应次之。严禁利用会道场所，设放缘金簿，或谈论是非。<br>    十、各祠堂对所收支款项及物资，均应记帐造册，按月按季列表，公布俾众周知，以昭公允。<br>    以上规章戒律，各祠（堂）务须共同遵守，切实执行。不得*废，若有违背者，视其情节轻重给予必要教育，过失严重的，即非本教门人，违法乱规，拥护政府，从严惩处。<br>
