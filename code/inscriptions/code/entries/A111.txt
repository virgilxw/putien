111  显应十境<br>    显应十境地处延寿陂北大沟，原属于仁德里的魏塘舖和洋池角铺。当地明代有魏塘村，清代分出郭庄、前庄、关前、后许、锄头埔、洋池角、厚亭、庄里、前港等村，现分属于显应、东墩二行政村。在神庙祭典中，主要按聚落分为东五境和西五境，东五境包括西港、庄厝、洋池、后亭、下洋，西五境包括魏塘、郭庄、东墩、埔前、下宇。<br>    位于魏塘村的显应庙（大社）为十境总宫，下洋村的显应祖社原为十境祖社，现为东五境祖社。<br>    仁德里显应庙，俗称“大社”。原有三殿，后改建为显应小学，1983年重建，1993年继续扩建，现为上下两殿，左右偏殿。上殿主神为姚公元帅（社公）、张氏夫人（社妈）、杨公太师，陪神为古传显灵三位中军（萧、倪、王）、圣子神、齐天大圣、魏公太保；左边殿为鲁府，供奉天前圣母、鲁部大神、鲁府姨妈、鲁部中军、五花童子；右边殿为弘仁祠，供奉黄大爷、李大爷、谢大爷、亚汉中军。每年农历正月初八至十六，姚公元帅、张氏夫人轮流到各村巡游；正月十七夜，诸神回宫，请道士做醮。农历四月杨公太师诞辰，各村轮流演戏庆贺；五月初五鲁部大神、鲁府姨妈诞辰，七月初七天前圣母诞辰，在显应庙演戏庆贺。六月初二姚公元帅诞辰，在显应庙演戏，巡游十境。八月张氏夫人诞辰，各村轮流演戏庆贺。新碑\2004显应十境显应庙碑.JPG、新碑\1785显应十境魏塘显应庙碑.JPG、新碑\2000显应十境魏塘显应庙碑.JPG、新碑\1836显应十境魏塘显应庙碑.JPG、新碑\1991显应十境魏塘显应庙碑.JPG、新碑\1991显应十境魏塘妈祖庙碑二.jpg<br><br><br><br>DSC05896<br>显应天后宫于乙酉年菊月默娘神凡纪念演戏二天收支公布如下！<br>收入：<br>二天缘金
