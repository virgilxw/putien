   智日纪念堂简介<br>    智日纪念堂为缅怀建筑木兰陂先贤智日和尚，于2001年建成。《八闽通志·高僧集》载：“北宋高僧智日，俗姓冯，法号智日，散性混俗，心有度世植德之举。”<br>    宋治平元年（公元1064年）至熙宁七年（公元1071年），长乐人钱四娘、林从世先后二次筑坡，因选址不当，不合水情，均告失败。宋熙宁八年（公元1075年），智日和尚领李宏实地堪察水情，吸取前者教训，慎选陂址，夕夜截溪插竹，似神指授，选址于河宽流缓的木兰山下（陂头村至屿上村港夹间的山麓），进行筑陂。在无水文、地质技术相助的情况下，智日和尚巧夺天工，悉心指导，于烂泥溪底上建成了一座巨型雄伟的闸式滚水坝。历时八年，大功告成。迄今已近千年，大陂利水莆阳，路人口碑不绝。1956年，国家水利部长傅作义莅临视察时，称智日为古代水利工程师。1962年，全国人大副委员长郭沫若至此，赋诗誉智日和尚，号神仙。1985年，日本古建筑专家组慕名前来进行科研活动，称智日禅师为佛门典范，有实地见证的“活济公”。海内外水利专家一致认为，早在千年前，智日和尚能用沙化基础，实乃举世无双的伟大创举。<br>
