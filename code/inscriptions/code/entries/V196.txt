船渡<br>    属珠坑行政村，现有人口约1000人，分为隆兴上社、隆兴下社、西胜祖社<br>三社。黄、刘为主要族姓。<br>    主要庙宇：<br>    古渡境龙凤宫、隆兴上下社，主神为尊主明王（二位）、后土夫人（二位）、<br>天上圣母；陪神为司马圣王、张公圣君、五路将军、五位帝爷、周公、百岁妈、<br>文武列圣。<br>    古渡境新兴宫、西胜祖社，主神为尊主明王、后土夫人；陪神为司马圣王、<br>天上圣母、张公圣君、五路将军、文武列圣。<br>    真武殿，主神为玄天上帝，陪神为玄坛元帅。<br>    中一堂，主神为三一教主。<br>    通明宫，主神为吕祖先师，陪神为武大人、黄大人、宋大人、宋将军、李中<br>军、文武列圣。<br>    元宵仪式：<br>    1．9，黄姓请新兴宫、西胜祖社诸神巡游本境，演戏，请道士做醮。由黄姓<br>组织，按丁口集资。<br>    1．10，刘姓请龙凤宫、隆兴上下社诸神巡游本境，演戏，请道士做醮。由<br>刘姓组织，按丁口集资。<br>    2．2，头福，演戏，请道士做醮。全村统一组织，按丁口集资。<br>    神诞庆典：<br>    3．13，武大人诞辰，演戏，请道士做醮。通明宫统一组织，自由捐资。<br>    3．23，天上圣母诞辰，演戏，请道士做醮。由刘姓组织，按丁口集资。<br>    7．7，黄大人诞辰，演戏，请道士做醮。通明宫统一组织，自由捐资。<br>    9．6，西胜祖社尊主明王、后土夫人诞辰，演戏，请道士做醮。由刘姓组织，<br>按丁口集资。<br>    11．13，宋大人诞辰，演戏，请道士做醮。通明宫统一组织，自由捐资。<br><br><br>
