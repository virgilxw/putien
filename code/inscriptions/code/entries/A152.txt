152  丰美七境<br>    丰美七境地处九里洋上沟，原属于永丰里的铁皂铺。当地明代有<br><br><br><br><br><br>DSC、下肖、方尾（丰美）等村，清代分出上邦尾、下邦尾、井头、中宅等村，现分属于丰美、丰山、厚峰三行政村。<br>    在<br><br><br><br><br><br>DSC村的石岑上，原有七境总宫，清嘉庆十六年因乌白旗冲突，所属各村由总宫分出，自建社、庙。现在各村社、庙的主神，都是原来从总宫分到的神灵：顶邦尾分中天大圣，下邦尾分刘公元帅，下肖分尊主明王、后土夫人，<br><br><br><br><br><br>DSC分齐天大圣，井头分法籙都统。中宅村是较迟形成的村落，故未分到总宫的神灵。<br>    在乌白旗械斗中，<br><br><br><br><br><br>DSC、下肖、井头、中宅为乌旗，下邦尾为白旗，顶邦尾为红旗。<br><br>
