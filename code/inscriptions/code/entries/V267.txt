侯厝<br>    现有人口约400余人，有陈、任、郑等姓，陈为主要族姓。<br>    主要庙宇：<br>    铁炉社，在板厝，主神为尊主明王、后土夫人。<br>    天清府，主神为田公元帅。<br>    聚源亭，主神为观音大士，陪神为苏大人、福德正神、大爷、二爷。。<br>    元宵仪式：<br>    1．9，到凌云殿进香、请斗。<br>    1．12，请龙应堂、铁炉社诸神巡游本村及桥头、下西，演戏，请道士做醮。<br>三村统一组织，按丁口集资。<br>    神诞庆典：<br>    2．12，温公元帅诞辰庆典，演戏，请道士做醮。五境统一组织，按丁口集<br>资。<br>    5．16，玉皇大帝诞辰庆典，到凌云殿进香。全村统一组织，按丁口集资。<br>    8．20，苏大人诞辰庆典，演戏，请道士做醮。全村统一组织，按丁口集资。<br>    10．1，尊主明王、后土夫人诞辰庆典，演戏，请道士做醮。铁炉社统一组<br>织，按丁口集资。<br><br>
