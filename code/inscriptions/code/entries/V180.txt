沟戈<br>    属呈山行政村，现有人口约1200人，有余、陈、朱、李等族姓，陈、余为<br>主要族姓。<br>    主要庙宇：<br>    莱登书社、名山宫，主神为尊主明王、后土夫人，陪神为天上圣母、黄府元<br>帅、司马圣王、鲁府大人、鲁府夫人、鲁府姨妈。<br>    元宵仪式：<br>    1．14-15， 请社炉到各户行道。<br>    1、16，请莱登书社诸神巡游本村，演戏，请道士做醮。全村统一组织，按<br>丁口集资。<br>   神诞庆典：<br>    2、25，参加呈祥宫齐天大圣巡游。<br>    3．23，天上圣母诞辰，演戏，请道士做醮。全村统一组织，按丁口集资。<br>    4．15，黄府元帅诞辰，演戏，请道士做醮。全村统一组织，按丁口集资。<br>    7．7，鲁府大神、鲁府夫人诞辰，演戏，请道士做醮。全村统一组织，按丁<br>口集资。<br>    8月卜日，司马圣王诞辰，演戏，请道士做醮。全村统一组织，按丁口集资。<br>    10月卜日，尊主明王、后土夫人诞辰，演戏，请道士做醮。全村统一组织，<br>按丁口集资。<br>    11月，兰盆普度，演戏，请道士做醮。全村统一组织，按丁口集资。<br><br>
