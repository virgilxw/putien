前墩<br>    全村人口700多人，生计除农业外，还有荔枝，泥木工，鞋革厂做工等。<br>    族姓：该村最早为张姓人，现为许和林。许原是张姓的长工，现张绝，许占全村人口四分之三，林占四分之一。<br>    据讲明朝时溪白有位方尚书，前墩有位许尚书同在朝为官，回乡时因要共引九华山下溪水，许尚书要引溪水灌溉农田，方尚书要引水冲茶，争执结果溪水到溪白只在石上打一个小孔，让少量的水流到溪白，溪水大都引一前墩来灌溉农田，因此事引起方尚书对许怀恨在心，把很多兵器抛在前墩后山的一个深潭中，到皇帝前奏讲，许尚书在家招兵买马，大造兵器，密谋造反，结果许尚书被除三族，一公田子出逃到今界外。许姓人在本村分为头房、二房、三房、四房，其中二房人数较多，原有祠堂。<br>    主要庙宇：<br>    中山孝友社，主神：尊主明王、后土夫人；陪神：田公元帅，黄、陈二殿将军，董、葛、周三将军。<br>    龙兴宫，主神：孔府大人；陪神：张公圣君、黄黑将军、大爷、二爷。<br>    聚沆亭，主神：观音大士。<br>    元宵仪式：<br>    1．9-10或1。11-12，与后墩轮流，请黄、陈二殿将军大小共4樽巡游本村，演戏，请师公1人做醮2夜。<br>    神诞庆典：<br>    3、19，6、19，9、19，观音大士诞辰庆典，各演戏2天。全村统一组织，按丁口集资。<br>    8．7 ，尊主明王、后土夫人诞辰庆典，演戏4天，请道士做醮。全村统一组织，按丁口集资。 <br>10．18，孔府大人诞辰庆典，演戏，请道士做醮，由蓝田组织，按丁口集资。，戏钱有部分题缘，部分按人均出。<br>    <br>
