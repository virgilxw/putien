   千年古刹上生寺，是唐代高僧妙应禅师的俗家胞弟本寂禅师舍田创建于唐咸通间（八六一至八七四），大顺元年（八九零）大儒黄璞又捐田入寺。宋银青光禄大夫林讽子吏部郎中林攸绩舍田贰佰陆拾余款，园地麦豆租叁拾石。寺内建有黄林两祠，南唐保大七年（九四九）修，元大德二年（一二九八）僧道潜重修，明景泰间（一四五零至一四五六）僧教修重建，僧廷芳，元迪建法堂，山门。弘治间（一四八八至一五零五）僧戒净重修。明正德间（一五零六至一五二一）寺内还创立诚书院。嘉靖四十一年（一五六二）毁于倭乱。万历十七年（一五八九）僧明聪重建。民国七年十一月被北洋军队烧毁。（一九九一）卓坡老人自发组成上生寺重建董事会在有关部门和十方大德的关心支持下在原址重建了大悲殿，大雄宝殿，地藏殿，妙莲阁，祖师，伽蓝殿及山门，天王殿亦正在筹建之中。上生寺建成为涵江增添一处文化旅游景点。<br><br>
