临水太后顺天圣母陈靖姑是妇女儿童的保护神，她与海上保护神林默娘齐。民间对其信仰从福建福州、闽江流域扩展到浙江、广东、台湾并流传海外，其庙宇遍布各地，厉前年而不衰，从民间传说到小说、戏剧、讲话、地方志，形成了民间文化，到今天仍有研究价值。<br>    陈靖姑不是神话塑造人物，而是确有其人，《闽都别记》，《福州府志》，《闽侯乡土志》，《台湾府志》和古田、罗源、长乐、闽侯、建宁、平潭等县志均有记载。<br>    陈靖姑是闽侯藤山下渡人（今局福州仓山区），出生于唐大历二年（767年）正月十五日，其父陈昌（任户部郎中），其母葛氏，陈昌偕妻兄葛如川，设馆教书，陈靖姑自幼聪慧，信奉观音大士，朝夕参拜，三岁能念观音经咒，七岁就读家馆，与古田刘杞莲同学，亲如兄妹，并由双方父母议约定婚，陈靖姑十四岁时，为避婚事，入“闾山大法院”学道，由许其君传授闾山正法和七星剑法，因不思拒学护胎救产法，学成别师下山，又因师徒情深，依依不舍，行廿四步回首看师父，故许真人叮嘱廿四岁不能动法品，终因不遵师嘱，脱胎祈雨而仙逝。<br>    陈靖姑伟大事绩主要有三个方面：<br>    （一）是除妖灭怪，保护人民。传说古田临水洞内白蛇精与长坑鬼勾结，将山洞变为华屋，蛇精变美女，诱青年入洞，加以杀害，适刘杞莲会文回家，路过此地，被骗入洞，逼婚不从欲加杀害，陈靖姑赶到，破洞救夫，并在双方家长百般要求下，念与杞莲数载同窗，患难相逢，遂改初衷，留在古田，与刘杞莲成婚，后将临水洞改为临水宫。白蛇精逃往福州皇宫，变为陈金凤皇后，真皇后被长坑鬼拖禁长坑山间，白蛇精在宫内残害三十六位宫娥，又迷惑闽王王廷钧，欲取陈靖姑之心以治病，靖姑将计就计，入宫除妖，将白蛇精斩为三段，分别镇在古田临水宫，井楼门七星井内，与龙潭角龙潭之中，并破长坑洞府，救回陈皇后，法炼三十六宫娥，闽王感德，救封为慈济临水陈夫人，并将三十六位宫娥赐予为徒，封为三十六婆，下令修建临水宫，陈靖姑还以闾山正法破袁广智茅山邪法，除灭蝴蝶精平定叛乱，避免屠城之祸，挽救将士百姓，闽王王磷甚为感德，复封陈靖姑为崇福临水陈夫人。陈靖姑在除妖灭怪过程中，还有其英明之处，即对顽固不化，一贯残害百姓者，如白蛇精、蝴蝶精、长坑鬼之类则以坚决消灭，而对愿志改邪归正者，如丹霞石猴、石夹二女、*婆等，则收归正果，协助除妖，受封丹霞大圣，石夹二夫人，*婆奶。<br>    （二）脱胎祈雨，拯救灾民；<br>    陈靖姑廿四岁时，福州大旱，田土分裂，稻禾枯萎，闽王令陈守元等道士设坛祈雨，因法力不够，越祈越晴，闽王发怒，责令五日内再祈无雨，则用柴塔烧死诸道士，陈守元无法，只好恳请胞妹陈靖姑祈雨，充分体现了陈靖姑之伟大精神：<br>    １、陈靖姑当时怀疑，按古规是不能参加神事活动的，她未学护胎救产法，脱胎祈雨，对自身十分危险；<br>    ２、她刚好廿四岁，许真人再三嘱咐廿四岁不能施动法器；<br>    ３、长坑鬼未除，会乘机陷害，在这自身十分危险的情况下，为了兄长守正及诸道士安全更是为了广大灾民，不顾自身安危，毅然答应施法祈雨，高尚精神深受万民爱戴，敬仰之所在。<br>    陈靖姑脱胎到白龙江龙潭角登台祈雨，左手执角，右手提剑，渡一片草席，到江中念动咒语，刹时乌云密布，普降大雨，黎民百姓，欢天喜地，当堂朝晖，可是陈靖姑脱放家中之胎，却被长坑鬼所吞食，突然胎慧血崩，肚痛力乏，同时草席下沉，定晴一看，乃是长坑鬼在水下拖草席，许真人知道徒见有难，立即抛石蛋变回复鸭母，拖住草席，方免遭水溺，但终因未学护胎救产之术，寒浸五腑六脏，致遭产死。<br>    （三）扶胎救产。相传陈靖姑升天后，英灵回到闾山，后悔当年不学护胎救产之法，许真人乃又传授护胎救产之法，故又成为民间儿童妇女之保护神，古时，医学落后，妇女生育视为“母难”，要请陈夫人（娘奶）庇佑，儿童十六岁以前，要“请奶过关”，清朝道光时，皇后难产，胎晃三日不下，太医束手无策，经闽人大臣奏请临水陈夫人救助，果然顺利分娩，母子均平安，道光皇帝大悦，敕封临陈太后。<br>
