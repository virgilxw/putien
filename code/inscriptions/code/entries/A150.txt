150  上方七境<br>    上方七境地处九里洋中沟，原属于永丰里的石庭新厝铺。当地明代有上方、上林、下刘、蔡墩等村，清代分化出石庭、东施、西施、埔埕、西刘、东蔡、西蔡、界外等村，现有人口约15000人。目前，石庭村已发展为相对独立的仪式系统，号称“石庭四甲”，而上方村也演变为石庭所属的角落。<br>    位于石东行政村的上方宫、福兴东社，原为上方七境的总宫，主祀二殿真君、尊主明王、后土夫人，陪祀护国王郎君、通天圣侯、惠济圣侯、刘公元帅、齐天大圣、欧氏仙妃、罗天大将、柳公圣侯、金公圣侯。民国时期，上方七境曾联合重修上方宫、福兴东社，并联合举行巡游和神诞、关戒仪式。2003年，石庭黄氏重建九鲤洞时，拆除了上方宫和福兴东社，也接收了所有神像。目前，原上方七境仅存上林、东施（西施）、埔城、西刘四村，并与原永丰里丰洋铺的东蔡、界下等村共同举行东岳注生大帝的巡游活动。<br>    在乌白旗械斗中，上方七境所属各村皆为白旗。<br><br>
