042  东山四社<br>    东山又称邹曾徐、三厝，地处后洋大沟，原属于兴福里的邹曾徐铺。当地明<br>代已有邹曾徐村，现有人口约10000人，分属于东山、东源二行政村。在神庙祭<br>典中，主要按聚落和姓氏分为上邹、下邹、东厝、郑厝四社，上邹的崇教宫为四<br>社总宫。<br>    本地主要有邹、郑二大姓。邹姓现有6000人左右，据族谱记载，于南宋建炎<br>年间（1127-1130）从山东邹县迁来莆田。明代邹姓为科举望族，共有2进士7举<br>人。现存藩参家庙，始建于明嘉靖二十九年（1550），为东山及新度、西园等地<br>邹氏的祖祠。新碑\1995东山四社藩参家庙碑.jpg、新碑\1995东山四社藩参家庙<br>碑二.jpg<br>    在乌白旗械斗中，东山四社皆为乌旗。<br><br>
