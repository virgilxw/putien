       昭灵宫董事会<br>                                                            二00六年三月初十日<br><br><br><br><br>DSC07962<br>昭灵宫碑志<br>    龙桥境昭灵宫始建于宋代哲宗五年（公元1102年），历史悠久。祀奉三位真君、王母娘娘。历经朝代沧桑，几经重修。宫宇拥有前后两殿厅、两厢护屋、宫埕戏台、郎亭台阁。雕梁画栋，五彩缤纷。建筑精致，宫貌巍峨幽静，气势宏伟，具有江南古建筑之艺术风格。历代四方香客游人纷至沓来，络绎不绝。每逢神灵寿诞时整天人来人往，灯火辉煌，香烟缭绕。是历代莆田西门外香火鼎盛的宫殿之一。可惜的是一九五八年发生大水灾，宫宇遭毁。1993年由四方虔诚人士热心踊跃捐资再建昭灵宫。2002年因龙桥桥头沟扩建需要昭灵宫再度重建。<br>    重建昭灵宫彩沿袭古建筑之艺术风格，结合现代高科技手段，宫宇重要部位增添了影雕图案和浮雕龙柱、月窗、麒麟。墙体改换花格碑、屋面使用琉璃瓦以及电光装璜设计等使整个宫貌显得更加巍峨肃穆壮观，更有气魄。<br>    重建后的昭灵宫位于莆田市府所在地，居民密集区和市场热闹区。可谓是黄金地段，风水宝地。今后将是四方善男信女文人墨客虔诚瞻仰参观游览的好去处，更是龙桥乡亲休闲娱乐的活动中心。<br>重建昭灵宫期间，有关单位领导对保护和发扬历史名胜古迹极为重视，不仅在经济上给予大力支持，而且在建筑上给予多方指导，加上虔诚人士慷慨解囊，热情捐资，使重建工程进展极为顺利，其公德无量。在此，表示衷心感谢，祝通缘者子孙昌盛，合家平安！<br>
