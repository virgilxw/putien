二十多年来，海内外乡亲齐心协力，乐捐巨资，重建新兴社，福惠堂，重修李氏宗祠，二度举行福惠堂持戒庆典，全村阖境平安，海内外厚施恩泽，同沾福祉，人丁兴旺，安居乐业。光阴似箭，日月如梭，上资持戒距今已十一年了，受戒新童均已长大成人，且为生涯之计，多数远赴异轮子邦劳务谋生。每逢元宵佳节，需向邻乡请求帮忙，近来，众乡亲纷纷建言提议再度持戒，充实新童力量，董事会集广大乡亲之意，为本宫社持续繁荣，祈求神明庇佑咱村海内外老幼平安，财丁骏发，择定于二00六年农历十月初六至十二日再度举行福惠堂持戒练童庆典。<br>    弘扬祖德，继往开来，整修族谱，势在必行。新墩李氏祖先自仲仁公开族以来，已有六百三十多年历史。为数众多的乡亲徒迁世界几十个国家和地区谋生，许多乡亲历经艰苦奋斗，事业有成，为所在国经济发展做出重大贡献。数百年来，咱村没有系统整过谱。根据海内外众族亲的提议，经各房代表商定，成立新墩李氏修谱理事会，认真探讨，深入研究，全面收集海内外新墩李氏宗亲资料，汇总成册。修谱由近代祖先查起，向上追记，查访历史资料，老人及古墓，这次修谱是本村开族以来的第一次全面修谱，乡亲散居几十个国家和地区，为求全面、准确，修谱工程非常艰巨，恳请海内外全体宗亲及时提供各种历史资料和现实人口资料。<br>    福惠堂再度持戒练童和新墩李氏首次全面修谱是咱村的特大喜事，工程巨大，费用甚多，敬请诸宗兄一如既往，鼎力支持。热心公益事业的宗兄宗婶将为世钦敬，千载流芳。为永久纪念在这次福惠堂持戒和新墩李氏修谱中捐资的海内外宗亲，经研究决定：<br>
