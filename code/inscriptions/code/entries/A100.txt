100  溪白十社<br>    溪白地处九华山麓，主要利用来自泗华陂的水源，原属于尊贤里的白杜铺。当地明代有白杜、溪埔、东坂等村，清代分出下尾、下溪、下刘、企溪等村，现有人口约5000人，统属于溪白行政村。在神庙祭典中，主要按聚落和姓氏分为十社。<br>    位于白杜村的孝友社为溪白祖社，位于路顶村的北极殿为溪白总宫。每年农历正月十五，溪白十社举行总元宵，请孝友社和北极殿诸神巡有所属各村。各社主持人按年龄大小轮流，孝友社主持人占卜而定。少数居民信基督教。<br>    本地居民主要为林、姚二大姓，另有沈、陈、吴、方、李等小姓。林姓约占溪白人口30%，主要分布在塘边、隔垄，据说是唐末入莆的，分君、子、居、利四房，原有祠堂，现改建为学校。姚姓主要分布在下头和汉头，约占溪白人口20%，分为乾、坤二房，坤房又分为仁、义、社、智四房，据讲是从涵江埔尾迁到溪白，祖墓在三山，有族谱。<br>    在本地历史上，白杜方氏为科举望族，明代共有3进士10举人。宋淳熙十四年（1187），为特科第一名方镐立“状元”牌坊。明景泰五年（1454），兴化知府为方述兄弟立“双桂”牌坊。明天顺四年（1460），为方逵立“进士”牌坊。明正统六年（1441），为方迪立“流庆”牌坊。<br>    在乌白旗械斗中，溪白十社皆为乌旗。<br><br>
