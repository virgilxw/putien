桥头<br>    现有人口200余人，有郭、卓等姓，无主要族姓。<br>    主要庙宇：<br>    铁炉社，在板厝，主神为尊主明王、后土夫人。<br>    天清府，主神为田公元帅。<br>    顺兴亭，主神为观音大士，陪神为温公元帅、韦驮尊天。<br>    元宵仪式：<br>    1．9，到凌云殿进香、请斗。<br>    1．12，请龙应堂、铁炉社诸神巡游本村及板厝、下西，演戏，请道士做醮。<br>三村统一组织，按丁口集资。<br>    神诞庆典：<br>    2．12，温公元帅诞辰庆典，演戏，请道士做醮。五境统一组织，按丁口集<br>资。<br>    8．15，温公元帅诞辰庆典，演木偶戏。本村组织，按丁口集资。<br>    9．2，观音大士诞辰庆典，演戏，请经师做醮。全村统一组织，按丁口集资。<br>    10．1，尊主明王、后土夫人诞辰庆典，演戏，请道士做醮。铁炉社统一组<br>织，按丁口集资。<br><br>
