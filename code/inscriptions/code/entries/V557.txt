     一代名将——张巡（司马圣王）简介<br>    张巡，祖籍蒲州河东，乃唐朝爱国名将。他年青时博览群术，通晓兵法，出仕后廉明清正，刚直不阿，官至副河南节度使，御史中丞。<br>    安史之乱期间，公元七五六年正月，张巡率军进守雍丘，与叛军激战，负伤数处，仍不退缩，当时叛军兵多达四万，而守军仅有三千，张巡分析敌强我弱，叛军必有轻敌之心，故制定了“出其不意、主动出击”的战略战策，夜劫粮草，草人借箭，偷袭敌营、智退叛军，一次次出奇制胜，取得了可喜的战绩，使令狐朔损兵折将，狼狈不堪，过后令狐朔老羞成怒，重整破旗，卷土而来，大兵至境，加上各地守军或纷纷投降或弃域遁退，为了保存实力，张巡被追放弃雍丘，退守睢阳。<br>    睢阳乃江淮门户，是军事要地。公元七五七年，叛军十余万人马围攻睢阳。张巡誓死坚守待授，全城上下同仇敌慷。连连挫败敌人。七月叛军再次围城。城中粮草却将尽，起初每一兵士每日还能分到一勺米，随后粮断草绝，便只能啃树皮、煮纸吃，捉天上的鸟雀，挖地下的老鼠，连抢甲、弓弩也被吃掉了。此时，外无授兵，内无粮草，万分无奈之际，张巡思痛杀掉爱妻，让兵士果腹，以示守城决心。将士们无不感动流泪，誓死跟随，可是由于四处求援无望，饥寒交追。十月的一天叛军又来攻城，城中将士再也没有力气站起来应战了，张巡眼含热泪，向西跪拜说：“睢阳这座孤城到了最后时刻，我生不能报效国家，死后再变厉鬼杀贼吧！”睢阳陷落，城中军民还活着的仅有四百余人，张巡被俘，坚贞不屈，慷慨就义，时年四十九岁，当唐大军赶到时，张巡等人已在三日前殉难了，十天后郭子仪等人收复了洛阳。<br> 夫碑者，乃纪事物于不朽也。霞楼村有太和庙焉，中祀大司马，张公雎阳，以其唐室元勋，四百战沙场。功赢郭李；但吁杨州都督，万千年浩气，永镇江淮。为其事功之彪炳，足以楷模于后世，故立庙以祀春秋，以彰其德。太和圣庙，历史悠久。据传圣庙始建厝后山坡，但以引渠之故，迁至今址，因有“先太和后太平”之说。当昔之时，雕楹刻桷，焕碧辉彤，宝殿瑶樨。炫光烂彩。怎奈风雨侵蚀，栋折梁朽，将有倒塌之危，乃先后于道光八年及公元四三年，两次重修；得以鲁殿灵光，巍然尚存。何期又经沧桑之变，致前座为墟，不堪回首。直至公元八四年，方得更修后座。令幸国泰民安，百业俱兴，同人等久盼恢复旧观，实冀弘扬忠义，期臻乡里之教化，以垂百代之风范，是以倡议重建谒诚修缮。幸得海外侨胞桑梓同人共解囊存，鼎力援建，尤其诸位筹建理事精心筹划，不辞劳苦，以致进程神速，时仅月余宏宇便告落成，真乃人工天助也，而令再睹新模之告竣，重瞻旧迹之焕彩，值此吉庆，深感信士仁人精诚可嘉；但愿合村老幼、历代子孙，长念建庙之初衷，不忘爱国之宗旨。谨立碑为志。<br>
