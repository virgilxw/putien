梧梓<br>    梧梓地处太平陂上圳，原属于兴教里的梧梓铺。当地明代有吴梓村，清代分为上梧梓、下梧梓、芦厝、坑尾等村，现有人口1000多人，统属于梧梓行政村。在神庙祭典中，主要社庙系统分为三社。<br>    本地居民以李姓最多，为李制干子孙，与洋尾李同宗，清明去洋尾祭祖，祖墓尚在。<br>    主要村庙：<br>    梧梓境重兴社，内分祖社、上社、下社，主神为尊主明王、后土夫人，陪神为玄天上帝、田公元帅、白牙将军、欧氏仙妃、齐天大圣、司马圣王、普济圣侯、柳金圣侯、文武诸神。<br>    本来重兴社又分出上社和下社，文革期间毁，现二社社公都集中于祖社，本尚未重兴。<br>    苍龙坛，主神为杨公太师，陪神为张公圣君、观音大士。<br>    元宵仪式：<br>    1．14-15，三社轮流请社炉到各户行道。<br>    1．16，全村跑香，请诸神香炉巡游本村。<br>    五、神诞庆典：<br>    2．1，尊主明王、后土夫人诞辰庆典，演戏2-3天，请道士坐醮，由全村按丁户集资。    <br>    2．10，天前圣母诞辰庆典，演戏二天，按人丁出缘金。<br>    2．25，齐天大圣诞辰庆典，演戏二天，按人丁出缘金。<br>    3．3，玄天上帝诞辰庆典，演戏二天，按人丁出缘金。<br>    3．9，金圣侯诞辰庆典，演戏二天，按人丁出缘金。<br>    4．9，田公元帅诞辰庆典，演戏二天，按人丁出缘金。<br>    4．26，诞辰庆典，演戏2-3天，请道士做醮，由全村按丁户出缘金。其他菩萨生日公烧香、设筵。<br>    5．5，鲁部大神诞辰庆典，演戏二天，按人丁出缘金。<br>    5．13，司马圣王诞辰庆典，演戏二天，按人丁出缘金。<br>    6．16，王天君诞辰庆典，演戏二天，按人丁出缘金。<br>    7．23，张公圣君诞辰庆典，演戏二天，按人丁出缘金。<br>    8．1，欧氏仙妃诞辰庆典，演戏二天，按人丁出缘金。<br>    8．20，普济圣侯诞辰庆典，演戏二天，按人丁出缘金。<br>    10．7，吉祥大将诞辰庆典，演戏二天，按人丁出缘金。<br>    10．11，柳圣侯诞辰庆典，演戏二天，按人丁出缘金。<br>    12．23，罗天大圣诞辰庆典，演戏二天，按人丁出缘金。<br><br><br>
